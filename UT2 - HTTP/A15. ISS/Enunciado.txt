Instala el servidor de IIS en una máquina virtual de windows
Establece como nombre de servidor www.nombreapellido.com. No uses ni tildes ni eñes. Modifica el fichero  C:/windows/system32/etc/hosts
Configura como directorio raíz para el servidor el directorio c:\inetpub\www\prueba2
Crea un alias para que al acceder al directorio http://www.nombreapellido.com/directorio1 se acceda al directorio ~/prueba1 (~ es el directorio de usuario actual)
Establece como puerto por defecto el 81, comprueba que funciona accediendo a alguna página y vuelve a dejarlo en el 80.
Establece como fichero por defecto juana.html y juan.html
Establece errores.log como fichero logs de errores. No se puede cambiar. Se ven en el visor de eventos.
Establece accesos.log como fichero logs de accesos.
Se deben mostrar índices automáticos
Al producirse un error 404 (Documento no encontrado) mostrar una página personalizada que indique que el documento no se ha encontrado. La página deberá contener al menos una imagen.
Crea dos host virtuales host1.nombreapellido.com y host2.nombreapellido.com. Los ficheros logs de accesos serán hostN_accesos.log (N será 1 y 2). Como fichero por defecto establecer indexN.html e indexN.php. En ambos mostrar índices de directorios. Añade estos nombres al fichero hosts de windows.
